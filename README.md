# spring-movie-app

En enkel Spring Boot-baserad Movie-webbapp som byggs ut stegvis med MVC, templates och datalager.

## Tech stack
- Spring Boot
- Spring MVC
- Thymeleaf
- Spring Data JPA
- Validation
- PostgreSQL
- Docker

## Starta projektet (Maven)
```bash
mvn spring-boot:run
```

## Status
Projektet är under utveckling.
